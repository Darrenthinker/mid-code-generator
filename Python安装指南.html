🐍 Python安装指南

⚠️ 重要提醒
安装过程中最关键的是勾选"Add Python to PATH"，否则无法在命令行中使用Python！

📥 第一步：下载Python

1. 访问官网

打开浏览器访问：https://www.python.org/downloads/

🔗 点击访问Python官网

2. 下载安装包

  • 点击页面上的大黄色按钮："Download Python 3.x.x"
  • 会自动检测您的操作系统并下载对应版本
  • 如果是Windows 64位系统，文件名类似：python-3.x.x-amd64.exe
  • 下载大小约25-30MB，请耐心等待

⚙️ 第二步：安装Python

🚨 关键步骤（必须执行）

  1. 双击下载的安装包运行
  2. 务必勾选"Add Python to PATH"（在安装界面底部）
  3. 选择"Install Now"进行快速安装
  4. 或选择"Customize installation"进行自定义安装
📌 为什么要勾选"Add Python to PATH"？
这样系统就能在任何位置找到Python命令，否则无法使用python命令启动服务器。

🔧 自定义安装选项（可选）

如果选择"Customize installation"，请确保勾选：

  • ✅ pip（Python包管理器）
  • ✅ tcl/tk and IDLE（Python开发环境）
  • ✅ Python test suite
  • ✅ py launcher
  • ✅ for all users（为所有用户安装）

高级选项中确保勾选：

  • ✅ Add Python to environment variables
  • ✅ Create shortcuts for installed applications
  • ✅ Add Python to PATH

✅ 第三步：验证安装

1. 打开命令行

  • 按Win + R键
  • 输入cmd按回车
  • 或者在开始菜单搜索"命令提示符"

2. 测试Python命令

在命令行中输入以下命令：

python --version

如果安装成功，应该显示类似：

Python 3.11.5

也可以尝试：

py --version

3. 测试pip（包管理器）

pip --version

应该显示类似：

pip 23.2.1 from ...

🚀 第四步：启动MID代码生成器

安装成功后，您可以：

  1. 方法一： 双击运行start.bat文件
  2. 方法二： 在项目目录打开命令行，运行：
    python -m http.server 3005
  3. 方法三： 使用Python启动器：
    py -m http.server 3005

成功启动后，浏览器访问：http://localhost:3005

🔧 常见问题解决

❌ "python不是内部或外部命令"

原因： 安装时没有勾选"Add Python to PATH"

解决方案：

  1. 重新运行Python安装程序
  2. 选择"Modify"修改安装
  3. 确保勾选"Add Python to PATH"
  4. 或者卸载后重新安装

❌ 端口被占用

如果3005端口被占用，可以使用其他端口：

python -m http.server 3006

然后访问：http://localhost:3006

❌ 权限问题

如果遇到权限问题，请：

  1. 以管理员身份运行命令提示符
  2. 或者安装时选择"为所有用户安装"

🎯 安装完成后的操作

1. 验证Python安装成功

确保命令行中python --version有正确输出

2. 启动MID代码生成器

双击start.bat或在命令行运行Python服务器

3. 开始使用

访问http://localhost:3005开始生成MID代码！

🐍 立即下载Python 📋 返回MID生成器

💡 小提示

  • Python安装包约25-30MB，下载需要几分钟
  • 安装过程约需要2-5分钟
  • 安装完成后可能需要重启命令行
  • 如果安装过程中遇到问题，可以选择"以管理员身份运行"
