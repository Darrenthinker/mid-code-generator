🚀 MID代码生成器启动指南

⚠️ 检测到Python未安装
您的系统似乎没有安装Python，请选择以下任一方法启动服务器：

📦 方法一：安装Node.js（推荐）

1. 访问 https://nodejs.org 下载并安装Node.js
2. 安装完成后，在项目目录打开命令行
3. 运行以下命令：
npm install
npm start
4. 浏览器会自动打开 http://localhost:3005

🐍 方法二：安装Python

1. 访问 https://www.python.org/downloads/ 下载Python
2. 安装时务必勾选 "Add Python to PATH"
3. 安装完成后，运行 start.bat 文件
4. 或者在命令行运行：
python -m http.server 3005

📁 方法三：使用Live Server扩展（VS Code）

1. 在VS Code中打开项目文件夹
2. 安装"Live Server"扩展
3. 右键点击index.html文件
4. 选择"Open with Live Server"

🌐 方法四：使用在线工具

1. 将所有文件上传到GitHub
2. 使用GitHub Pages或Netlify部署
3. 获得在线访问链接

📱 方法五：直接打开文件（简单但功能受限）

1. 直接双击打开 index.html 文件
2. 会在浏览器中打开（但某些功能可能不可用）
注意：此方法可能导致某些功能（如文件上传）无法正常工作

✅ 成功启动后

无论使用哪种方法，成功启动后您将看到：

  • MID Code Generator & Validator 主页面
  • 三个主要标签：Generate MID Code, Validate MID Code, Batch Processing
  • 智能解析和手动输入两种模式
🚀 直接尝试打开（方法五） 📦 下载Node.js 🐍 下载Python
